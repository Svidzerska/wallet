How to start:
1. clone repository;
2. install node modules (npm install);
3. start: npm run dev; (It concurrently started "npm run server" "npm run client");

Technologies: 
HTML5 and CSS3, SASS
JavaScript ES6+
TypeScript
React.js & Redux (CRA, React Hooks, react-number-format)
REST API
node.js
Webpack

Реалізований функціонал: 
Інтерактивний гаманець, за допомогою якого користувач може слідкувати за своїми
фінансами. Є можливість додавання будь-якої кількості банківських карток, зі зберіганням
інформації на сервері (в проекті також - використання бази данних, наприклад MongoDB).
Інформація щодо карток, що вводить користувачем, верифікується, зокрема за допомогою
добре відомих алгоритмів (як от алгоритм Луна). Картки виводяться на екран в візуально
зручному вигляді. Є можливість видаляти картки. Також, алгоритм розраховує загальний баланс 
по карткам і виводить в окремому вікні, що є привабливою функцією для користувача. Проект також 
передбачає розширення функціоналу для можливості редагувати картки, а також вносити на віртуальний 
рахунок готівку. Ці функції на разі не реалізовані виключно через обмеженість у часі, і 
необхідність надати користувачам мінімальний функціонал вже сьогодні. Слідкуйте за оновленнями!
